                                               Usage:
                                               -------------------

                                                m - Start

                                                h - Throw ball

                                                'Left' - Strafe Ball Left

                                                'Right' - Strafe Right

                                                y - Increase Throw Velocity

                                                v - Decrease Throw Velocity

                                                u - Increase Throw Angle

                                                b - Decrease Throw Angle

                                                q - Exit

                                                w/a/s/d - Rotate View 
  
                                                '+/-' - Zoom in/out
